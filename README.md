# LearnMate – Agentic AI for Personalized Course Pathways

## Overview
LearnMate is an AI-powered learning coach that helps students discover personalized learning paths based on their interests, current skills, and career goals. It leverages Agentic AI to provide adaptive course recommendations, dynamic learning roadmaps, and continuous progress tracking.

## Domain
Education (EdTech)

## Technologies Used
- IBM Granite Model
- IBM watsonx.ai
- IBM Cloud Lite
- LangFlow
- Retrieval-Augmented Generation (RAG)
- FAISS/Chroma Vector Database
- FastAPI
- Streamlit

## Features
- Personalized Course Recommendations
- Dynamic Learning Roadmaps
- Skill Gap Analysis
- Career Guidance
- Progress Tracking Dashboard
- Multi-Agent AI Architecture

## Repository Contents
- app.json
- ProblemStatement.pdf
- ProjectPresentation.pptx

## Objective
To provide students with an AI-powered personalized learning assistant that recommends the right courses, tracks learning progress, and guides them toward their career goals.
